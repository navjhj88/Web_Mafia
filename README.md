# Web_Mafia
웹으로 동작하는 텍스트 기반 마피아게임입니다.



## 프로그램 구성

### Story

1. 로그인
   - 로그인 하면 방 대기실로 들어감
2. 방 만들기
   - 방제 적기
   - 인원수 제한(8~12)
   - 비밀번호
3. 방 참가
   - 사람수 및 게임진행여부 검사
   - 채팅
   - 준비버튼
   - 다른 유저 프로필에 준비상태 확인
   - 방 나가기
   - 방 삭제여부 확인
4. 방 나가기
   - 방 대기실로 가기
   - 방 채팅방에서 삭제
